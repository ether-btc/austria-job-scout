### 1. Extracted `_stats()` Helper Function

**Before:**
```python
mean_lat = statistics.mean(latencies_ms)
median_lat = statistics.median(latencies_ms)
stdev_lat = statistics.stdev(latencies_ms) if len(latencies_ms) > 1 else 0
min_lat = min(latencies_ms)
max_lat = max(latencies_ms)
```

**After:**
```python
def _stats(values: list[float]) -> dict[str, float]:
    """Calculate statistics for a list of values."""
    if not values:
        return {"mean": 0, "median": 0, "stdev": 0, "min": 0, "max": 0}
    return {
        "mean": statistics.mean(values),
        "median": statistics.median(values),
        "stdev": statistics.stdev(values) if len(values) > 1 else 0,
        "min": min(values),
        "max": max(values),
    }

# Usage:
s = _stats(latencies_ms)
```

**Savings:** ~8 lines per call site (2 call sites = 16 lines)

### 3. Removed Per-Iteration Sample Collection

**Before:**
```python
results = []
for i, text in enumerate(texts, 1):
    # ... benchmark code ...
    results.append({"iteration": i, "latency_ms": round(elapsed_ms, 2)})

summary = {
    # ... other fields ...
    "samples": results,
}
```

**After:**
```python
# No results list - just print per-iteration progress
for i, text in enumerate(texts, 1):
    # ... benchmark code ...
    print(f"  [{i}/{iterations}] {elapsed_ms:6.1f} ms")

summary = {
    # ... other fields (no "samples") ...
}
```

**Rationale:** Per-iteration samples were only used for verbose JSON output. Summary stats are sufficient for performance validation.

**Savings:** 4 lines + cleaner output
